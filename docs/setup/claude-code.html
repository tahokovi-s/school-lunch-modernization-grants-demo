Setup guide

Claude Code App Installed And Signed In

Use this page to confirm that the Claude Code desktop app opens and is connected to the Claude account you plan to use during the session. No terminal or command-line check is required for this workshop.

What Success Looks Like

Claude Code is ready when the desktop app launches, shows you as signed in, and lets you open the Code workspace for a folder.

Success means you can open Claude Code, reach the Code tab or coding workspace, and type a short message such as Confirm you are ready for the STAX training. Do not change files.

If You Are On macOS

  1. Open the official Claude Code desktop guide using the link below.
  2. Download the macOS desktop app from the official guide.
  3. Open the installer and follow the prompts.
  4. Launch Claude from the Applications folder, Launchpad, or Spotlight Search.
  5. Sign in with the Claude account you will use for the workshop, then open the Code tab or coding workspace.

If You Are On Windows

  1. Open the official Claude Code desktop guide using the link below.
  2. Download the Windows desktop app from the official guide.
  3. Run the installer and open Claude from the Start menu.
  4. Sign in with the Claude account you will use for the workshop.
  5. Open the Code tab or coding workspace. If the app says the plan needs an upgrade, use the access guide.

If Claude Code Does Not Open

Use the desktop app path for this workshop. The official docs also describe CLI options, but the live session will not ask participants to run terminal commands.

  1. Confirm that you downloaded Claude Code from the official Claude Code desktop guide.
  2. Close Claude and reopen it from Applications on macOS or the Start menu on Windows.
  3. Confirm that the sign-in browser uses the same Claude account you plan to use during the workshop.
  4. If the Code tab is blocked by plan access, open the access guide from the setup checklist.
