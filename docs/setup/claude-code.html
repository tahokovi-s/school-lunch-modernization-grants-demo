Setup guide

Claude Code Installed And Signed In

Use this page to confirm that Claude Code opens from a terminal and is connected to the Claude account you plan to use during the session.

What Success Looks Like

If Claude Code is ready, this command prints a version number instead of an error:

claude --version

A successful result may look like 2.1.89, claude 2.1.89, or another version number. Any current version is fine for the workshop.

If You Are On macOS

  1. Open Spotlight Search with Command-Space.
  2. Type Terminal, then press Return.
  3. Copy claude --version, paste it into Terminal, and press Return.
  4. If you see a version number, Claude Code is installed.
  5. To confirm sign-in, type claude and press Return. Sign in with your Claude account if prompted.

If You Are On Windows

  1. Open the Start menu.
  2. Type PowerShell, then open Windows PowerShell.
  3. Copy claude --version, paste it into PowerShell, and press Enter.
  4. If you see a version number, Claude Code is installed.
  5. To confirm sign-in, type claude and press Enter. Sign in with your Claude account if prompted.

If The Command Is Not Found

If the terminal says that claude is not recognized, Claude Code is probably not installed yet or the terminal cannot find it.

  1. Open the official Claude Code setup page.
  2. Follow the instructions for macOS or Windows. On Windows, the official docs explain both native Windows and WSL options.
  3. Close and reopen Terminal or PowerShell after installing.
  4. Run claude --version again.
