Terminal Version Checks

A terminal is just a text box for your computer. You type one command, press Enter or Return, and read what comes back. For this workshop, the goal is only to confirm that four commands print version numbers.

The Four Checks

Run each command one at a time. After each one, press Enter or Return.

codex --version claude --version git --version python3 --version

Success means each command prints some kind of version number. The exact numbers do not need to match anyone else's.

If You Are On macOS

  1. Open Spotlight Search with Command-Space.
  2. Type Terminal, then press Return.
  3. Click inside the Terminal window.
  4. Copy the first command from this page, paste it into Terminal, and press Return.
  5. Repeat for the other three commands.

If You Are On Windows

  1. Open the Start menu.
  2. Type PowerShell, then open Windows PowerShell.
  3. Click inside the PowerShell window.
  4. Copy the first command from this page, paste it into PowerShell, and press Enter.
  5. Repeat for the other three commands.

If Something Says Not Found

  • codex not found: open the Codex setup guide from the checklist.
  • claude not found: open the Claude Code setup guide from the checklist.
  • git not found: install Git from the official Git page.
  • python3 not found: install Python from the official Python page. On Windows, also try python --version; if that works, mention it during setup.
