Setup guide

Terminal Checks Are Not Required

This page is kept for anyone who bookmarked an older setup guide. The Thursday workshop now uses the Codex and Claude Code desktop apps directly, so participants do not need to run terminal or command-line checks before the session.

What To Do Instead

Use the setup checklist in the main guide. Confirm that both apps open, you are signed in, account access works, and no project folder has been created yet.

Success means Codex and Claude Code open as apps and are ready for the live session. The facilitator will handle any command-line details from inside the agent apps if they are needed.

If You Are On macOS

  1. Open Codex from Applications, Launchpad, or Spotlight Search.
  2. Open Claude from Applications, Launchpad, or Spotlight Search.
  3. Confirm each app is signed in with the account you plan to use.
  4. Return to the setup checklist and mark the app checks complete.

If You Are On Windows

  1. Open Codex from the Start menu.
  2. Open Claude from the Start menu.
  3. Confirm each app is signed in with the account you plan to use.
  4. Return to the setup checklist and mark the app checks complete.

If You Need Help

Use the app-specific setup pages instead of troubleshooting terminal commands.
