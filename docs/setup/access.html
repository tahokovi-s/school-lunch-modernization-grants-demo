Setup guide

Confirm Account Access

Use this page to confirm that your accounts can actually run the Codex app and the Claude Code app. Installation alone is not enough if the account is not signed in or the plan does not include the tool.

What To Confirm

  • You can sign in to ChatGPT with the account you will use for Codex.
  • You can sign in to Claude with the account you will use for Claude Code.
  • Your access is paid, institution-provided, or otherwise confirmed to include the coding-agent apps.
  • You are not relying on a trial or free account unless the official product page says it includes the access you need.

Plan names, prices, and usage limits change. Use the official product pages as the source of truth before paying for anything.

If You Are Using A Personal Account

  1. Open the ChatGPT pricing page and confirm your plan includes enough Codex access for the workshop.
  2. Open the Claude plan page and confirm your plan includes Claude Code access.
  3. Use the same account when each desktop app asks you to sign in.
  4. If your browser opens a sign-in page from either app, do not rush. Check that the email address shown is the account you mean to use.

If You Are On macOS

  1. Open Safari, Chrome, or another browser.
  2. Sign in to ChatGPT and Claude using the accounts you plan to use for the workshop.
  3. Open the Codex app and confirm it uses the same ChatGPT account.
  4. Open the Claude Code app and confirm it uses the same Claude account.

If You Are On Windows

  1. Open Edge, Chrome, or another browser.
  2. Sign in to ChatGPT and Claude using the accounts you plan to use for the workshop.
  3. Open the Codex app and confirm it uses the same ChatGPT account.
  4. Open the Claude Code app and confirm it uses the same Claude account.

If You Are Using University Or Organization Access

  1. Check whether your institution provides a ChatGPT, OpenAI, Claude, Anthropic, Team, Edu, Enterprise, or similar account.
  2. Sign in with the institution-approved email address.
  3. If an app opens a browser sign-in page, choose the institution account if prompted.
  4. If you are not sure which account to use, bring that question to the setup portion of the session.
