Setup guide

Codex App Installed And Signed In

Use this page to confirm that the Codex app opens and is connected to the ChatGPT account you will use during the workshop. No terminal or command-line check is required for this session.

What Success Looks Like

Codex is ready when the app launches, shows you as signed in, and lets you start or open a coding-agent session.

Success means you can type a short message in Codex, such as Confirm you are ready for the STAX training. Do not change files., and Codex can respond.

If You Are On macOS

  1. Open the official Codex page using the link below.
  2. Choose the macOS download from the official page.
  3. Open the downloaded installer or app file and follow the prompts.
  4. Launch Codex from the Applications folder, Launchpad, or Spotlight Search.
  5. Sign in with the ChatGPT account you will use for the workshop.

If You Are On Windows

  1. Open the official Codex page using the link below.
  2. Choose the Windows option from the official page or Microsoft Store link.
  3. Install Codex and open it from the Start menu.
  4. Sign in with the ChatGPT account you will use for the workshop.
  5. If Windows asks for permission to run the app, allow it only if the installer came from the official OpenAI link.

If Codex Does Not Open

Do not switch to a terminal workaround for this workshop. Use the official product page first, then bring the issue to the setup portion of the session if it is still stuck.

  1. Confirm that you downloaded Codex from the official OpenAI page.
  2. Close Codex and reopen it from Applications on macOS or the Start menu on Windows.
  3. Confirm that the browser sign-in window uses the same ChatGPT account you plan to use during the workshop.
  4. If the app asks for access to a folder, choose a simple folder only when the facilitator asks you to during the session.
