Setup guide

Codex Installed And Signed In

Use this page if you are not sure whether Codex is installed or if the word Terminal feels unfamiliar. You only need to confirm that Codex opens and is connected to your ChatGPT account.

What Success Looks Like

If Codex is ready, this command prints a version number instead of an error:

codex --version

A successful result may look like codex 0.131.0 or another number. The exact number is not important for the workshop.

If You Are On macOS

  1. Open Spotlight Search with Command-Space.
  2. Type Terminal, then press Return.
  3. Copy codex --version, paste it into Terminal, and press Return.
  4. If you see a version number, Codex is installed.
  5. To confirm sign-in, type codex and press Return. If a browser sign-in appears, sign in with the ChatGPT account you will use for the session.

If You Are On Windows

  1. Open the Start menu.
  2. Type PowerShell, then open Windows PowerShell.
  3. Copy codex --version, paste it into PowerShell, and press Enter.
  4. If you see a version number, Codex is installed.
  5. To confirm sign-in, type codex and press Enter. If a browser sign-in appears, sign in with the ChatGPT account you will use for the session.

If The Command Is Not Found

If Terminal or PowerShell says that codex is not recognized, Codex is probably not installed yet or the terminal cannot find it.

  1. Open the official Codex CLI setup page.
  2. Use the current install instructions for your operating system.
  3. Close and reopen Terminal or PowerShell after installing.
  4. Run codex --version again.
