Setup guide

Do Not Create The Project Folder Yet

The live session will create the workshop folder together. Please do not create, clone, or rename a folder before Thursday unless you are prompted during setup.

What Success Looks Like

The apps are installed and signed in, but the workshop folder does not exist on your machine yet. During the live walkthrough, you will create a folder named My_RA_Tasks in a sensible place in your file home, then open that folder in Codex as the project root.

Success means you arrive ready to use Codex and Claude Code, then create My_RA_Tasks during the live walkthrough.

If You Are On macOS

  1. Do not make a folder in Finder for this workshop yet.
  2. Do not download a project zip or raw-data zip before the live walkthrough.
  3. Do not ask Codex to create the project before the session.
  4. It is fine to leave Codex and Claude Code closed after setup until Thursday.

If You Are On Windows

  1. Do not make a folder in File Explorer for this workshop yet.
  2. Do not download a project zip or raw-data zip before the live walkthrough.
  3. Do not ask Codex to create the project before the session.
  4. It is fine to leave Codex and Claude Code closed after setup until Thursday.

If You Already Made One

No harm done. Leave it alone for now and mention it during setup. You may be asked to rename it or choose a fresh location so your live walkthrough matches the group.
