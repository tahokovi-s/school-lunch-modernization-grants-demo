Setup guide

Do Not Create The Project Folder Yet

The live session starts from a realistic inbox handoff: first the assignment email, then the project folder. Please do not create, clone, or rename a folder before Thursday unless the facilitator asks.

What Success Looks Like

The apps are installed and signed in, but the Hollywood film tax credit project folder does not exist on your machine yet.

Success means you arrive ready to use Codex and Claude Code, and everyone creates the same project together during the live walkthrough.

If You Are On macOS

  1. Do not make a folder in Finder for this workshop yet.
  2. Do not download a zip file of the project yet.
  3. Do not ask Codex to create the project before the session.
  4. It is fine to leave Codex and Claude Code closed after setup until Thursday.

If You Are On Windows

  1. Do not make a folder in File Explorer for this workshop yet.
  2. Do not download a zip file of the project yet.
  3. Do not ask Codex to create the project before the session.
  4. It is fine to leave Codex and Claude Code closed after setup until Thursday.

If You Already Made One

No harm done. Leave it alone for now and mention it during setup. The facilitator may ask you to rename it or choose a fresh location so your live demo matches the group.
