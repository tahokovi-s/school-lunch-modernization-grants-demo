Codex Skills

This is optional material for the end of the session. The core research workflow does not require installing skills before Thursday.

What Success Looks Like

Codex should recognize the official skills catalog, name the optional workshop skills, and explain what it would change before it modifies any user-level or Global Codex settings.

https://github.com/openai/skills

Success means Codex installs only the optional workshop skills that are not already available, asks before changing settings, and tells you to restart Codex afterward.

Optional Workshop Skills

  • define-goal: turns a fuzzy research handoff into a concrete objective and success criteria.
  • openai-docs: helps Codex use current OpenAI and Codex documentation when product details matter.
  • pdf: helps with research papers, policy documents, scanned notices, and layout-sensitive PDFs.
  • jupyter-notebook: helps scaffold or clean up exploratory notebooks for later empirical work.

The official repo includes many other skills. Do not install deployment, Figma, GitHub PR, Notion, security, media, or browser-automation skills for this workshop unless the optional exercise explicitly asks for them.

Prompt To Give Codex

Copy this into Codex only during the optional skills exercise:

Use $skill-installer with https://github.com/openai/skills as the official skills catalog. Install only this optional workshop set if the skills are not already available: define-goal, openai-docs, pdf, and jupyter-notebook. Do not install any other skills. Before changing any user-level or Global Codex settings, explain exactly what you will change and wait for my confirmation. After installation, tell me to restart Codex and show me how to confirm the skills are available.

Only use skills from repositories you trust. For this workshop, use the official OpenAI repository above unless the live session gives you a different link.

If You Are On macOS

  1. Open the Codex app.
  2. Start a new Codex session with no project folder selected, unless the app requires one.
  3. Paste the prompt from this page into Codex only if the group is doing the optional skills exercise.
  4. Read Codex's explanation before allowing it to change user-level or Global Codex settings.
  5. If you confirm an install, close and reopen Codex afterward so the new skills are available.

If You Are On Windows

  1. Open the Codex app from the Start menu.
  2. Start a new Codex session with no project folder selected, unless the app requires one.
  3. Paste the prompt from this page into Codex only if the group is doing the optional skills exercise.
  4. Read Codex's explanation before allowing it to change user-level or Global Codex settings.
  5. If you confirm an install, close and reopen Codex afterward so the new skills are available.

If Codex Cannot Install A Skill

  1. Ask Codex to run /skills or open the skills picker inside the app.
  2. Ask Codex whether any selected skill is already built in or already installed.
  3. If Codex says it cannot install skills, check the official Codex skills documentation.
  4. Bring the issue to the setup portion of the session if it still does not work.
