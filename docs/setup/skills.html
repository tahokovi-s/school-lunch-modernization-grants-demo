Setup guide

Open /skills In Codex

Skills are reusable instructions that help Codex handle specific workflows. For this session, you only need to confirm that you can open the skills menu or ask Codex about skills.

What Success Looks Like

When Codex is open, typing this should show skill-related options or information:

/skills

Success means Codex recognizes the slash command and shows a skills list, menu, or explanation. The exact screen may differ by Codex version.

If You Are On macOS

  1. Open Terminal.
  2. Type codex and press Return.
  3. Wait for Codex to open.
  4. Type /skills and press Return.
  5. If Codex shows a skills menu or message, this checklist item is complete.

If You Are On Windows

  1. Open PowerShell.
  2. Type codex and press Enter.
  3. Wait for Codex to open.
  4. Type /skills and press Enter.
  5. If Codex shows a skills menu or message, this checklist item is complete.

If /skills Does Not Open

  1. Run codex --version to confirm Codex is installed.
  2. If Codex opens but does not recognize /skills, check the official Codex skills documentation.
  3. If you recently installed or updated Codex, close the terminal and open it again.
  4. Bring the issue to the setup portion of the session if it still does not work.
