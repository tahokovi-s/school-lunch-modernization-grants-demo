Setup guide

Add Skills After Codex Works

Do this only after Codex is installed, opens, and is signed in. The goal is to give Codex the official OpenAI skills repository link, then ask Codex to help add only the skills needed for the workshop.

What Success Looks Like

Codex should recognize the official skills catalog and explain what it would add before it changes your Global Codex settings.

https://github.com/openai/skills

Success means Codex can list relevant skills from the official repo, explain where they would be installed, and confirm that any selected skill is available after restart.

Prompt To Give Codex

Copy this into Codex after Codex is already open:

Use https://github.com/openai/skills as the official skills catalog. List skills that may help this workshop. Do not install anything yet. First explain what you would add to my Global Codex settings and wait for my confirmation.

Only use skills from repositories you trust. For this workshop, use the official OpenAI repository above unless the facilitator gives a different link.

If You Are On macOS

  1. Open Terminal.
  2. Type codex and press Return.
  3. Wait for Codex to open.
  4. Paste the prompt from this page into Codex.
  5. Read Codex's explanation before allowing it to change Global Codex settings.
  6. If you confirm an install, restart Codex afterward so the new skills are available.

If You Are On Windows

  1. Open PowerShell.
  2. Type codex and press Enter.
  3. Wait for Codex to open.
  4. Paste the prompt from this page into Codex.
  5. Read Codex's explanation before allowing it to change Global Codex settings.
  6. If you confirm an install, restart Codex afterward so the new skills are available.

If Codex Cannot Add Skills

  1. Run codex --version to confirm Codex is installed.
  2. Open Codex again with codex.
  3. Ask Codex to open or explain /skills.
  4. If Codex says it cannot install skills, check the official Codex skills documentation.
  5. Bring the issue to the setup portion of the session if it still does not work.
