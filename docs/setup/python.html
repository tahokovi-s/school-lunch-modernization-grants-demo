Python Installed And Ready For Codex

Use this page to confirm that Python is installed and that Codex can run a small Python check for the workshop. Participants do not need to type command-line checks themselves.

What Success Looks Like

Codex can find Python, run a tiny Python script, report the Python version, and confirm that it can execute code for the STAX training.

Success means Codex reports that Python is available and that a small script ran correctly.

Prompt To Give Codex

Copy this into Codex during Module 1:

Please check whether Python is installed and available to you. If it is available, run a tiny Python script that prints the Python version and confirms that 2 + 2 equals 4. Then tell me whether you can run Python code for the STAX training. Do not modify any project files.

If Python Is Missing

If Codex says Python is missing or cannot run code, install Python from an official or institution-approved source, then reopen Codex and try the check again.
